Sports Ticker Angular Directive Demo

A responsive (screen width > 767px) approximation of a sports news ticker, built with HTML5 and Angular.js.

NOTE: Ticker does NOT display on mobile phones (this is intentional), and performance is not optimized for non-desktop experiences.

This was mainly just a fun side-project for me to teach myself more about CSS animations and Angular.
